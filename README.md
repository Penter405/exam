## 請遵守以下規則:
### 1.請將main.py 檔案 與 data 資料夾 放在同一個路徑,也就是它們的上層資料夾相同
### 2.請勿擅自編輯任何檔案。如果你想要看筆記，可以去data 資料夾找
### 3.刷題請使用main.py 檔案
